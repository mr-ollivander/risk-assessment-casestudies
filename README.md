## Methodology

Each assessment follows a structured framework:
- **Asset identification** — what has value and needs protecting
- **Threat identification** — what event or actor could cause harm
- **Vulnerability identification** — what weakness enables the threat
- **Likelihood & Impact rating** — qualitative, evidence-based
- **Risk rating** — derived from a likelihood × impact matrix
- **Response** — proportionate, sequenced, tied to the specific vulnerability
